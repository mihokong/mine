# Minesweeper

의존성 없이 동작하는 정적 지뢰찾기 게임입니다.

## 실행

브라우저에서 `index.html`을 열면 바로 실행됩니다.

## GitHub Pages 배포

1. 이 디렉터리를 GitHub 저장소에 push합니다.
2. GitHub 저장소의 `Settings` → `Pages`로 이동합니다.
3. `Build and deployment`에서 `Deploy from a branch`를 선택합니다.
4. `Branch`를 `main` 또는 사용하는 기본 브랜치로 선택하고, 폴더는 `/ (root)`로 둡니다.
5. 저장하면 `https://사용자명.github.io/저장소명/` 주소로 배포됩니다.

## 조작

- 왼쪽 클릭: 칸 열기
- 오른쪽 클릭: 깃발 표시
- 열린 숫자 칸 클릭: 주변 깃발 수가 숫자와 같으면 주변 칸 열기
